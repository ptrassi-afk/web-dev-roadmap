# Link to my GitHub account

In developing exercise 04 – Links, I used some fundamental elements to correctly structure the HTML document:

- 'head': responsible for containing structural information about the document that is not directly displayed to the user.
- 'meta': used to provide metadata about the page, such as encoding settings and information relevant to browsers and search engines.
- 'link': used to establish the connection between the HTML file and external resources, such as stylesheets (CSS).
- 'body': corresponds to the part of the document that is displayed to the user, that is, all the content visible on the page.
- 'a': used to create clickable and interactive links, allowing navigation between pages or resources.

Additionally, a link to the GitHub account was created using the href attribute to define the destination address and the target attribute so that the link opens in a new browser tab.
